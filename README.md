
TaskFlow – Project Management Dashboard

TaskFlow is a modern project management dashboard built with React, TypeScript, and Vite. It helps users organize, track, and manage projects efficiently through a clean and responsive interface.

The application focuses on scalable component architecture, state management, and persistent client-side storage to simulate real-world productivity tools.

🚀 Features
Project Lifecycle Management: Create, edit, update, and delete projects with real-time UI updates
Search & Filtering: Quickly search projects for improved usability and productivity
Status Tracking: Track project progress using structured status categories (e.g., Active, Completed, Pending)
Dashboard Analytics: Visual overview of project distribution and status breakdown
Persistent Storage: Uses Local Storage to retain data across sessions without backend dependency
Responsive Design: Fully responsive UI optimized for desktop, tablet, and mobile devices
Reusable Component Architecture: Modular design for scalability and maintainability

🛠️ Tech Stack
React
TypeScript
Vite
Tailwind CSS (if used, add it here)
Local Storage API

📦 Architecture Overview
Component-based structure for scalability
Centralized state management using React hooks
Separation of UI, logic, and data layers
Optimized rendering for performance
